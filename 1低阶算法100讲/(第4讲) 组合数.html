一、概念定义

  这一节，我们简单了解一下组合数。数学上的组合数是用公式实现的，而程序实现过程中，为了避免乘法产生的溢出，一般采用的是递推的形式。

1、组合数定义

  组合数表示为 CnmC_n^mCnm​，它的含义是：从 nnn 个不一样的元素中，选择 mmm 个元素的方案数。

2、组合数递推公式

  组合数的递推公式如下：Cnm=Cn−1m−1+Cn−1mC_n^m=C_{n−1}^{m-1} + C_{n−1}^mCnm​=Cn−1m−1​+Cn−1m​

3、递推公式的理解

  可以理解为：含特定元素的组合有 Cn−1m−1C_{n−1}^{m-1}Cn−1m−1​，不含特定元素的排列为Cn−1mC_{n−1}^mCn−1m​。
  更加具体的，含有特定元素 111 的情况下，则需要从剩下 n−1n-1n−1 个元素中，再找 m−1m-1m−1 个元素，即方案数为 Cn−1m−1C_{n−1}^{m-1}Cn−1m−1​；而不含有特定元素 111 的情况下，则是从剩下的 n−1n-1n−1 个元素中，再找 mmm 个元素，即方案数为 Cn−1mC_{n−1}^mCn−1m​。
  而这两种情况就代表了所有情况，加和就是原问题的方案数。

4、举例说明

  从 1，2，3，4，5 (n=5)(n = 5)(n=5) 中取出 2 (m=2)(m = 2)(m=2) 个元素的组合 CnmC_n^mCnm​：12  13  14  15  23  24  25  34  35  4512 \ \ 13 \ \ 14 \ \ 15 \ \ 23 \ \ 24 \ \ 25 \ \ 34 \ \ 35 \ \ 4512  13  14  15  23  24  25  34  35  45
  这些组合中要么含有元素 “1”，要么不含。

1）含有1

12  13  14  1512 \ \ 13 \ \ 14 \ \ 1512  13  14  15 把里面的 “1” 都去掉，得到：2  3  4  52 \ \ 3 \ \ 4 \ \ 52  3  4  5 等价于从 2，3，4，5 (n−1)(n−1)(n−1) 中取出1 (m−1)(m−1)(m−1) 个元素的组合。

2）不含有1

23  24  25  34  35  4523 \ \ 24 \ \ 25 \ \ 34 \ \ 35 \ \ 4523  24  25  34  35  45 等价于从 2，3，4，5 (n−1)(n−1)(n−1) 中取出 2 (m)(m)(m) 个元素的组合。

  总方案数等于上面两种情况方案数之和，即 Cnm=Cn−1m−1+Cn−1mC_n^m=C_{n−1}^{m-1}+C_{n−1}^mCnm​=Cn−1m−1​+Cn−1m​。

5、边界处理

  考虑当 nnn 和 mmm 都等于 0 时，方案数就是一；而当 n=mn=mn=m 时，方案也只有一种。

二、题目描述

  给定一个非负整数 nnn，生成 「 杨辉三角 」 的前 nnn 行。在 「 杨辉三角 」 中，每个数是它左上方和右上方的数的和。

四、源码剖析

  以下是C语言版本，如果对 「 二级指针 」 还没有概念，建议使用 C++ 的 vector 。

int** generate(int numRows, int* returnSize, int** returnColumnSizes){
    int i, j;
    int **ret = (int **)malloc(numRows * sizeof(int *));       // (1)
    *returnSize = numRows;                                     // (2)
    *returnColumnSizes = (int *)malloc(numRows * sizeof(int)); // (3)
    for(i = 0; i < numRows; ++i) {
        ret[i] = (int *)malloc( (i+1) * sizeof(int) );         // (4)
        (*returnColumnSizes)[i] = i+1;                         // (5)
        for(j = 0; j < i+1; ++j) {
            if(j == 0 || j == i) {
                ret[i][j] = 1;                                 // (6)
            }else {
                ret[i][j] = ret[i-1][j] + ret[i-1][j-1];       // (7)
            }
        }
    }
    return ret;
}
  • (1)(1)(1) 申请一个长度为numRows的二维数组，第二维是一个整型指针；
  • (2)(2)(2) 定义第一维的长度为numRows；
  • (3)(3)(3) 定义第二维的长度，由于第二维的每个元素长度不等，所以这里需要用一个数组；
  • (4)−(5)(4)-(5)(4)−(5) 申请第二维的内存空间，长度为 i+1i+1i+1；
  • (6)(6)(6) ret[i][j]ret[i][j]ret[i][j] 代表了 CijC_i^jCij​，处理边界情况；
  • (7)(7)(7) 递推公式；
