四、源码剖析

#define ll long long
ll f(ll a, ll b, ll c) {
    if (b == 0)
        return 1 % c;           // (1)
    ll v = f(a*a % c, b/2, c);  // (2)
    if (b % 2)
        v = v * a % c;          // (3)
    return v;
}
  • (1)(1)(1) 考虑 b=0b=0b=0 的情况；
  • (2)(2)(2) 考虑 bbb 为非零偶数的情况；
  • (3)(3)(3) 考虑 bbb 为奇数的情况；
