二、题目描述

  给你一个整数数组 nums，其中 nums[i]表示第 i个袋子里球的数目。同时给你一个整数 maxOperations。
  你可以进行如下操作至多 maxOperations次：
  选择任意一个袋子，并将袋子里的球分到 2 个新的袋子中，每个袋子里都有 正整数 个球。比方说，一个袋子里有 5 个球，你可以把它们分到两个新袋子里，分别有 1 个和 4 个球，或者分别有 2 个和 3 个球。
  你的开销是单个袋子里球数目的 最大值 ，你想要 最小化 开销。请你返回进行上述操作后的最小开销。

三、算法详解

  (1)(1)(1) 前期思路总结：
    (1.1)(1.1)(1.1) 考虑所有袋子里的球的数目最大值为 mmm，那么，如果一开始不做任何操作，它的开销就是 mmm；
    (1.2)(1.2)(1.2) 但凡做了一次操作，都可以将 mmm 进行减小，从而使得开销减小（如果有两个 mmm 就不一定了）。
    (1.3)(1.3)(1.3) 每次操作一定是选择最大的进行减小，而且是尽量平均分配。
    (1.4)(1.4)(1.4) 从上文可知，操作数为 xxx，袋子中球数目最大值为 f(x)f(x)f(x)，f(x)f(x)f(x) 是 xxx 的单调不增函数，于是，可以二分 f(x)f(x)f(x)，求出 xxx 来判断是否满足 x <= maxOperations。
  (2)(2)(2) 算法尝试：
    (1.1)(1.1)(1.1) 二分一个答案 TTT；
    (1.2)(1.2)(1.2) 对于任意一个袋子，里面的球数为 xxx，那么将它分成如下形式：
TTTTTTTtT T T T T T TtTTTTTTTt
    其中 ttt 的范围就是 [0,T)[0, T)[0,T)，那么就是要求分成了多少堆，再减一。这里如果想不清楚，可以用数学归纳法，或者 待定系数法，将T=3T = 3T=3，分别取 x=0，1，2，3，4，5...x = 0，1，2，3，4，5 ...x=0，1，2，3，4，5...，得到球数为 xxx，分成最大为 TTT 的堆的操作次数为：⌊x−1T⌋\lfloor \frac {x - 1}{T} \rfloor⌊Tx−1​⌋
    (1.3)(1.3)(1.3) 将所有的这些值相加，如果小于等于maxOperations，则二分成立，缩小区间，继续枚举。

四、源码剖析

class Solution {
    bool check(vector<int>& nums, int max, int total) {
        int ans = 0;
        for(int i = 0; i < nums.size(); ++i) {
            int x = nums[i];
            // max + max + max + max + [0, max) = x
            ans += (x - 1) / max;
            if(ans > total) {
                return false;
            }
        }
        return true;
    }
public:
    int minimumSize(vector<int>& nums, int maxOperations) {
        int l = 1;
        int r = 1000000000;
        int ans = 0;
        while(l <= r) {
            int mid = (l + r) >> 1;
            if( check(nums, mid, maxOperations) ) {
                ans = mid;
                r = mid - 1;
            }else {
                l = mid + 1;
            }
        }
        return ans;
    }
};
