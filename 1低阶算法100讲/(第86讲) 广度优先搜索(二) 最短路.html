一、概念定义

  最短路问题分为如下几种：
    (1)(1)(1) Dijkstra：单源最短路，时间复杂度 O(n2)O(n^2)O(n2)；
    (2)(2)(2) Bellman-Ford：单源最短路，可判定负权圈，时间复杂度 O(nm)O(nm)O(nm)，并且是 差分约束 的基础；
    (3)(3)(3) Floyd：多源最短路，时间复杂度 O(n3)O(n^3)O(n3)；
    (4)(4)(4) Dijkstra + Heap：加堆优化后，对于完全图来说，时间复杂度 O(nlogn)O(nlogn)O(nlogn)；
    (5)(5)(5) SPFA：不断松弛，时间复杂度还是 O(nm)O(nm)O(nm)，但是比 Bellman-Ford 更加高效；
  其中 (4)(4)(4) 和 (5)(5)(5) 是利用 广度优先搜索 来实现的。
  普通广搜问题，任意两点间的边权是一样的，所以一旦找到一定是最短路；当任意两点间边权不一样时，就只能通过 SPFA 来不断松弛，等队列为空以后才能最终找到最短路。

二、题目描述

  有 n(n≤100)n(n \le 100)n(n≤100) 个城市，按从 000 到 n−1n-1n−1 编号。给你一个边数组 edges，其中 edges[i]=[fromi,toi,weighti]edges[i] = [from_i, to_i, weight_i]edges[i]=[fromi​,toi​,weighti​] 代表 fromifrom_ifromi​ 和 toito_itoi​ 两个城市之间的双向加权边，距离阈值是一个整数 distanceThreshold。
  返回能通过某些路径到达其他城市数目最少、且路径距离 最大 为 distanceThreshold的城市。如果有多个这样的城市，则返回编号最大的城市。
注意，连接城市 iii 和 jjj 的路径的距离等于沿该路径的所有边的权重之和。

三、算法详解

  (1)(1)(1) 对于这个问题，需要求出每个城市到其它城市的距离小于阈值的城市的个数，并且取个数最少，当个数相同的时候，取编号最大的。
  (2)(2)(2) 求每个城市到其它城市满足小于阈值，可以采用 SPFA或者Dijkstra + Heap优化。
  (3)(3)(3) 由于这题的数据量很小，所以不需要优化，直接SPFA即可。

四、源码剖析

class Solution {
    #define maxn 110
    #define inf -1

    int Min(int a, int b) {            // (1)
        if(a == inf) return b;
        if(b == inf) return a;
        return a < b ? a : b;
    }

    int mat[maxn][maxn];


    int spfa(int n, int u, int dt) {    // (2)
        int i;
        queue<int> q;
        int dist[maxn];
        memset(dist, inf, sizeof(dist));
        dist[u] = 0;
        q.push(u);

        while(!q.empty()) {
            u = q.front();
            q.pop();
            if(dist[u] > dt) {
                continue;
            }
            
            for(i = 0; i < n; ++i) {
                if(mat[u][i] == inf) {
                    continue;
                }
                int todist = dist[u] + mat[u][i];
                if(dist[i] == inf || todist < dist[i]) {
                    dist[i] = todist;
                    q.push(i);
                }
            }
        }
        int cnt = 0;
        for(i = 0; i < n; ++i) {
            if(dist[i] != inf && dist[i] <= dt) {
                ++cnt;
            }
        }
        return cnt;
    }

public:
    int findTheCity(int n, vector<vector<int>>& edges, int dt) {
        int i, j;
        int retId, retCnt;
        memset(mat, inf, sizeof(mat));

        for(i = 0; i < edges.size(); ++i) {// (3)
            int u = edges[i][0];
            int v = edges[i][1];
            int w = edges[i][2];
            mat[u][v] = mat[v][u] = Min(mat[u][v], w);
        }
        retCnt = 111;
        for(i = n-1; i >= 0; --i) {        // (4)
            int cnt = spfa(n, i, dt);
            if(cnt < retCnt) {
                retCnt = cnt;
                retId = i;
            }
        }
        return retId;
    }
};
  • (1)(1)(1) 定义一个最小值函数；
  • (2)(2)(2) int spfa(int n, int u, int dt)表示总共n个结点，从第 u号结点出发，最大距离为dt，能够到达的结点数目；
  • (3)(3)(3) 用邻接矩阵存储距离关系；
  • (4)(4)(4) 遍历每个点，计算和它相邻，距离小于等于给定值的个数，取最小值；
