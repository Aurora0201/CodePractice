二、题目描述

  给你一个 rows x cols大小的矩形披萨和一个整数 k，矩形包含两种字符： 'A'（表示苹果）和 '.'（表示空白格子）。你需要切披萨 k-1次，得到 k块披萨并送给别人。
  切披萨的每一刀，先要选择是向垂直还是水平方向切，再在矩形的边界上选一个切的位置，将披萨一分为二。
  如果垂直地切披萨，那么需要把左边的部分送给一个人；
  如果水平地切，那么需要把上面的部分送给一个人。
在切完最后一刀后，需要把剩下来的一块送给最后一个人。请你返回确保每一块披萨包含 至少 一个苹果的切披萨方案数。由于答案可能是个很大的数字，请你返回它对 109+710^9 + 7109+7 取余的结果。

三、算法详解

  (1)(1)(1) 暴力切，横着切，竖着切；
  (2)(2)(2) 利用 int dfs(int k, int xl, int xr, int yl, int yr)函数表示对于 [(xl,xr)→(yl,yr)][(x_l, x_r) \to (y_l, y_r)][(xl​,xr​)→(yl​,yr​)] 这个矩形切成 kkk 块且满足每一块都有至少一个苹果的方案数。
  (3)(3)(3) 于是，可以枚举横着切，竖着切，判断一个区域内是否有苹果，可以利用 二维前缀和。
  (4)(4)(4) 由于题目限制，切出来的矩形它的右下角是固定的，所以我们可以用一个 dp[k][x][y]数组来表示切成 kkk 块，并且左上角在原图的坐标为 (x,y)(x, y)(x,y) 的方案数。
  (5)(5)(5) 记忆化搜索求解。

四、源码剖析

int sum[51][51];
int dp[11][51][51];

#define mod 1000000007

void init(char ** pizza, int n) {               // (1)
    int m = strlen(pizza[0]);
    int i, j;

    for(i = 1; i <= n; ++i) {
        for(j = 1; j <= m; ++j) {
            sum[i][j] = (pizza[i-1][j-1] == 'A' ? 1 : 0);
        }
    }

    for(i = 1; i <= n; ++i) {
        for(j = 1; j <= m; ++j) {
            sum[i][j] += sum[i-1][j] + sum[i][j-1] - sum[i-1][j-1];
        }
    }
}

// (xl, yl) - (xr, yr)
int getSum(int xl, int xr, int yl, int yr) {   // (2)
    return sum[xr][yr] - sum[xl-1][yr] - sum[xr][yl-1] + sum[xl-1][yl-1];
}

int dfs(int k, int xl, int xr, int yl, int yr) {
    if(k == 1) {
        return getSum(xl, xr, yl, yr) ? 1 : 0;
    }
    if(dp[k][xl][yl] != -1) {
        return dp[k][xl][yl];                   // (3)
    }
    int ans = 0;
    for(int i = xl; i < xr; ++i) {              // (4)
        // (xl, i, yl, yr) + (i+1, xr, yl, yr)
        if(getSum(xl, i, yl, yr)) {          
            ans += dfs(k-1, i+1, xr, yl, yr);
            if(ans >= mod) {
                ans -= mod;
            }
        }

    }

    for(int i = yl; i < yr; ++i) {              // (5)
        // (xl, xr, yl, i) + (xl, xr, i+1, yr)
        if(getSum(xl, xr, yl, i)) {
            ans += dfs(k-1, xl, xr, i+1, yr);
            if(ans >= mod) {
                ans -= mod;
            }
        }
    }

    dp[k][xl][yl] = ans;

    return ans;
}

int ways(char ** pizza, int pizzaSize, int k){
    int n = pizzaSize;
    int m = strlen(pizza[0]);
    init(pizza, pizzaSize);
    memset(dp, -1, sizeof(dp));
    return dfs(k, 1, n, 1, m);
}

  • (2)(2)(2) 容斥原理求矩阵部分和；
  • (3)(3)(3) 记忆化的过程；
  • (4)(4)(4) 横着切；
  • (5)(5)(5) 竖着切；
