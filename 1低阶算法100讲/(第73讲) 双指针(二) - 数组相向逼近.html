一、概念定义

  数组相向逼近问题，一般就是定义两个指针，从数组的两端开始，不断的去check是否满足给定条件，根据不同条件，来选择是左指针自增，还是右指针自减，最终求出问题的解。

二、题目描述

  给你一个整数数组 nums和一个整数 k。每一步操作中，你需要从数组中选出和为 k的两个整数，并将它们移出数组。返回你可以对数组执行的最大操作数。

三、算法详解

  (1)(1)(1) 首先，对数组进行排序；
  (2)(2)(2) 定义左指针为l = 0，右指针为r = n-1；
  (3)(3)(3) 根据两个指针对应的值相加得到val，分三种情况讨论：
    (3.1)(3.1)(3.1) 如果val > k，则右指针自减；
    (3.2)(3.2)(3.2) 如果val < k，则左指针自增；
    (3.3)(3.3)(3.3) 如果val = k，计数器加一，右指针自减，左指针自增；
  (4)(4)(4) 最后返回计数器；

四、源码剖析

int cmp(const void*a, const void*b) {
    return *(int *)a - *(int *)b;
}

int maxOperations(int* nums, int numsSize, int k) {
    int l = 0;
    int ans = 0;
    int r = numsSize - 1;
    qsort(nums, numsSize, sizeof(int), cmp);
    while(l < r) {
        int val = nums[l] + nums[r];
        if(val > k) {
            --r;
        }else if(val < k) {
            ++l;
        }else {
            ++l, --r;
            ++ans;
        }
    }
    return ans;
}
