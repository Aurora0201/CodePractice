一、概念定义

  深度优先搜索，就是穷举所有情况，可能会伴随一些剪枝。组合问题，就是给定 nnn 个元素，选择其中 mmm 个元素，方案总数是 CnmC_n^mCnm​，所以如果枚举所有方案，这是一个指数级的时间复杂度。
  为什么是指数级呢？
  因为：
2n=Cn0+Cn1+...+Cnn−1+Cnn2^n = C_n^0 + C_n^1 + ... + C_n^{n-1} + C_n^n2n=Cn0​+Cn1​+...+Cnn−1​+Cnn​
  可以想象成：一个物品取或者不取，并且加上一些限制：所有取得的物品总数是 mmm。
  那么，对于这样的一类问题，是 NP 问题，如果求的是方案数，那就是背包问题，利用动态规划求解；如果求的是某种方案，或者某几种方案，那就是搜索问题，可以利用深度优先搜索求解。

二、题目描述

  给定两个整数 nnn 和 kkk，返回范围 [1,n][1, n][1,n] 中所有可能的 kkk 个数的组合。你可以按 任何顺序 返回答案 (n≤20)(n \le 20)(n≤20)。

三、算法详解

  (1)(1)(1) 通过观察数据范围，2202^{20}220 不大，所以 C20kC_{20}^{k}C20k​ 也是能够接收的范围，所以可以直接对某个物品进行选或者不选，统计总共选出来的物品是否到达 kkk 个，从而枚举所有可行解。

四、源码剖析

class Solution {
    vector<int> stk;                                 // (1)
    vector<vector<int>> ans;                         // (2)

    void dfs(int l, int r, int k) {                  // (3)
        int i;
        if(k == 0) {
            ans.push_back(stk);                      // (4)
            return ;
        }
        for(i = l; i <= r; ++i) {
            stk.push_back(i);                        // (5)
            dfs(i+1, r, k-1);                        // (6)
            stk.pop_back();                          // (7)
        }
    }

public:
    vector<vector<int>> combine(int n, int k) {
        stk.clear();
        dfs(1, n, k);                                //(8)
        return ans;
    }
};
  • (1)(1)(1) vector<int> stk代表的是深度优先搜索时候用到栈；
  • (2)(2)(2) vector<vector<int>> ans代表的是最后的结果数组；
  • (3)(3)(3) void dfs(int l, int r, int k)代表的是从 [l,r][l, r][l,r] 这个区间内，选择 kkk 个数的方案，并且会将方案最终存储到 ans。
  • (4)(4)(4) 递归出口。也就是说从 [l,r][l, r][l,r] 里面取 000 个数，那么，当然栈中的元素就是最后的答案，直接放入结果数组。
  • (5)(5)(5) 表示第 iii 个选了的情况，并且将它放进栈中；
  • (6)(6)(6) dfs(i+1, r, k-1);表示从 [i+1,r][i+1, r][i+1,r] 选择剩下 k−1k-1k−1 的方案；
  • (7)(7)(7) 表示第 iii 个不选，所以需要从栈中取出来；
  • (8)(8)(8) dfs(1, n, k)表示从 [1,n][1, n][1,n] 选择 kkk 个的所有方案。
