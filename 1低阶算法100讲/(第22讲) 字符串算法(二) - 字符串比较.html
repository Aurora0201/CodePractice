  这一节，我们来讲一下C语言中最简单的字符串相关算法，字符串的比较。

一、概念定义

  通过上一节 《算法零基础100讲》(第21讲) 字符串算法(一) - 字符串遍历 的学习，我们已经知道了字符串就是一种特殊的数组，这一节，我们来了解几个常用的字符串函数

1、字符串拷贝

  字符串拷贝是字符串操作中非常常用的手段，即将一个字符串赋值给另一个字符串，在C语言中，调用的系统函数为strcpy。实现如下：

char src[10] = "Hello";
char tar[10];
strcpy(tar, src);

  这个函数调用完毕，字符串tar的值就变成了"Hello"；

2、字符串比较

  同样常用的，还有字符串比较，在C语言中，调用的系统函数为strcmp。简单实现如下：

char src[10] = "Hello";
char tar[10] = "hello";
int x = strcmp(tar, src);
int y = strcmp(tar, "hello");

  以上语句调用完毕后，由于 C语言是大小写敏感的，所以 xxx 的值 非零 表示字符串比较不相等，而 yyy 的值 为零 表示字符串比较相等。

二、题目描述

  实现一个函数，对于字符串数组，找出给定字符串的位置。

三、算法详解

  利用线性枚举进行遍历。线性枚举算法，我们之前在 《算法零基础100讲》(第17讲) 线性枚举(一) 已经学过，这里无非是枚举到对应字符串时，和给定字符串进行比较判等。

四、源码剖析

int find(char** words, int s, char* ss){ // (1)
    int i;
    for(i = 0; i < s; ++i) {             // (2)
        if(!strcmp(words[i], ss)) {      // (3)
            return i;                    // (4)
        }
    } 
    return -1;                           // (5)
}
  • (1)(1)(1) 字符串是数组，字符串数组自然就是二维数组，所以需要用**表示，即二级指针；
  • (2)(2)(2) 线性枚举所有字符串；
  • (3)(3)(3) 给定字符串 和 枚举到的字符串 进行判等操作；
  • (4)(4)(4) 相等则返回对应下标；
  • (5)(5)(5) 一个都没有找到，则直接返回 -1；
