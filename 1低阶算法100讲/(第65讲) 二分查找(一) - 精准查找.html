一、概念定义

1、二分查找概述

  二分查找解决的是 单调函数 上的查找问题。然后就有人问了，我遇到的二分查找都是在数组中找一个数，这个也是函数吗？
  广义地来说，数组就是一些离散的点，所以它是一种 离散函数。所以，数组元素的查找其实也是在函数中进行查找。如下图所示，代表的是一个五个元素的数组：

2、二分查找本质

  对于一个单调函数 y=f(x)y = f(x)y=f(x) 而言，如果 xxx 已知，则能够通过函数定义，直接求出 yyy；而 yyy 已知，如何得到 xxx ？
  这就是二分查找要干的事情。yyy 已知，如何得到 xxx 的问题如果抽象成数组查找问题，其实就是：当一个数组元素的值已知的时候，查找它的下标。也就是 x=f−1(y)x = f^{-1}(y)x=f−1(y)，即 反函数的概念。
  当然，在我们实际编码过程中，不需要去思考反函数的概念。
  于是，我们开始了解下下面的 蓝色点 的 数组下标 如何求解？

3、二分查找原理

  所有的单调函数判定都是雷同的，这里用严格单调递增函数 f(x)f(x)f(x) 来举例（其它单调函数同理可得），要求在这个函数中，找到值为 vvv 的 自变量 xxx 的值（也就是图中蓝色点的 xxx 坐标）。
  首先，有两个游标 lll 和 rrr 分别指向 函数 的 定义域 的两个端点。也就是 [l,r][l, r][l,r] 这个区间就是函数 y=f(x)y = f(x)y=f(x) 中 xxx 的取值范围。
  然后，取定义域的中点 mid=l+r2mid = \frac {l + r} { 2}mid=2l+r​，分三种情况讨论：
    (1)(1)(1) f(mid)=vf(mid) = vf(mid)=v，则直接返回 midmidmid；
    (2)(2)(2) f(mid)>vf(mid) > vf(mid)>v，说明要查找的 vvv，一定在 midmidmid 的左边，也就是区间 [l,mid−1][l, mid-1][l,mid−1] 中；
    (3)(3)(3) f(mid)<vf(mid) < vf(mid)<v，说明要查找的 vvv，一定在 midmidmid 的右边，也就是区间 [mid+1,r][mid+1, r][mid+1,r] 中；

  于是，转换成了求更小的子区间的问题。

4、二分查找实现

  我们把这件事情抽象出来，用一个函数binarySearch(arr, l, r, v)来表示在数组 arr的区间 [l,r][l, r][l,r] 上找值为 vvv 的下标，如果没有找到，返回 -1。

int f(int *arr, int x) {
    return arr[x];
}

int binarySearch(int *arr, int l, int r, int v) {
    if(l > r) {
        return -1;                              // (1)
    }
    if(l == r) {
        return arr[l] == v ? l : -1;            // (2)
    }
    int mid = (l + r) >> 1;                     // (3)
    int fv = f(arr, mid);                       // (4)
    if(fv < v) {
        return binarySearch(arr, mid+1, r, v);  // (5)
    }else if(fv > v) {
    	return binarySearch(arr, l, mid-1, v);  // (6)
    }
    return mid;                                 // (7)
}
  • (1)(1)(1) 空区间，一定无法找到对应的值；
  • (2)(2)(2) 只有一个数的区间，对应的值如果相等返回下标，否则返回 -1；
  • (3)(3)(3) 对区间进行折半；
  • (4)(4)(4) 计算折半点对应的函数值fv；
  • (5)(5)(5) 如果fv < v，说明不可能在区间 [l,mid][l, mid][l,mid] 之间，所以递归查找 [mid+1,r][mid+1, r][mid+1,r] 这个区间；
  • (6)(6)(6) 如果fv > v，说明不可能在区间 [mid,r][mid, r][mid,r] 之间，所以递归查找 [l,mid−1][l, mid-1][l,mid−1] 这个区间；
  • (7)(7)(7) fv == v时直接返回mid；
