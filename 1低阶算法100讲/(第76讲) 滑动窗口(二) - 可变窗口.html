二、题目描述

  给定一个字符串 s，请你找出其中不含有重复字符的 最长子串 的长度。

三、算法详解

  (1)(1)(1) 首先，定义一个滑动窗口 [i,j][i, j][i,j]，初始化是一个空窗，也就是 i=0,j=−1i = 0, j = -1i=0,j=−1；
  (2)(2)(2) 然后，不断向右移动滑动窗口的右边界，同时记录当前字符的出现次数，可以用一个哈希表来记录；
  (3)(3)(3) 当第 jjj 个字符在哈希表中出现超过 1 次，则移动滑动窗口的左边界，同时更新对应字符的出现次数，注意，这里需要循环迭代计算；
  (4)(4)(4) 当右边界枚举到整个字符串结尾，则结束计算。

四、源码剖析

int lengthOfLongestSubstring(char * s){
    int n = strlen(s);
    int hash[256];
    int max = 0;
    int i = 0, j = -1;
    memset(hash, 0, sizeof(hash));      // (1)
    while(j < n - 1) {                  // (2)
        ++j;
        ++hash[ s[j] ];                 // (3)
        while( hash[ s[j] ] > 1 ) {     // (4)
            --hash[ s[i] ];
            ++i;
        }
        if(j - i + 1 > max) {           // (5)
            max = j - i + 1;
        }
    }
    return max;
}
  • (1)(1)(1) 清空字符哈希表；
  • (2)(2)(2) 滑动窗口右边界为字符串结尾；
  • (3)(3)(3) 对哈希表对应字符自增；
  • (4)(4)(4) 当第 jjj 个字符在哈希表中出现超过 1 次，则移动滑动窗口的左边界，同时更新对应字符的出现次数，注意，这里需要循环迭代计算；
  • (5)(5)(5) 随时记录满足条件的最大值；
