二、题目描述

  给定三个正整数 aaa、bbb 和 ccc。你可以对 aaa 和 bbb 的二进制表示进行位翻转操作，返回能够使按位或运算 a∣b==ca | b == ca∣b==c 成立的最小翻转次数。
「位翻转操作」是指将一个数的二进制表示任何单个位上的 1 变成 0 或者 0 变成 1 。

四、源码剖析

int getbit(int v, int i) {
    return (v & (1<<i)) ? 1 : 0;    // (1)
}

int minFlips(int a, int b, int c){
    int i;
    int ans = 0;
    int ai, bi, ci;
    for(i = 0; i < 31; ++i) {
        ai = getbit(a, i);
        bi = getbit(b, i);
        ci = getbit(c, i);
        if( (ai | bi) == ci ) {     // (2)
            continue;
        }else if(ci == 0) {
            ans += (ai + bi);       // (3)
        }else if(ci == 1) {
            ans += 1;               // (4)
        }
    }
    return ans;
}
  • (1)(1)(1) 返回一个数的第 iii 个二进制位是 0 还是 1；
  • (2)(2)(2) 满足 (a | b == c)则无需翻转；
  • (3)(3)(3) 根据 aaa 和 bbb 中的 1 的数量来决定要翻转几次；
  • (4)(4)(4) 这种情况下，势必 aaa 和 bbb 都为 0，所以只要挑一个变成 1 即可。
