一、概念定义

  容斥原理也是高中数学的东西，我们学过集合，对于两个集合 AAA 和 BBB 。如何求他们的并集的元素个数呢？

1、两个集合的并

  集合 AAA 与 BBB 的并集的元素个数，等于集合 AAA 的元素个数与集合 BBB 的元素个数的和，减去集合 AAA 与 BBB 的交的元素个数，即：∣A∪B∣=∣A∣+∣B∣−∣A∩B∣|A∪B|=|A|+|B|-|A∩B|∣A∪B∣=∣A∣+∣B∣−∣A∩B∣
  画个图容易理解，这里 ∣A∩B∣|A∩B|∣A∩B∣ 由于被计算了两次，所以需要减去。

二、题目描述

  给定 nnn、aaa、bbb、ccc，求不大于 nnn，且能被 aaa 或 bbb 或 ccc 整除的数有多少个，其中 1≤n≤1091 \le n \le 10^91≤n≤109，1≤a×b×c≤1091 \le a \times b \times c \le 10^91≤a×b×c≤109。

三、算法详解

  能被 aaa 和 bbb 同时整除的个数记为 ababab，能被 bbb 和 ccc 同时整除的记为 bcbcbc，能被 aaa 和 ccc 同时整除的记为 acacac，能被 aaa bbb ccc 同时整除的个数，记为 abcabcabc。根据容斥原理，能被三个数中任意一个数整除的个数为 a+b+c−ab−ac−bc+abca+b+c - ab - ac - bc + abca+b+c−ab−ac−bc+abc
  而不超过 nnn，且能被 aaa 和 bbb 同时整除的个数为 nlcm(a,b)\frac {n} {lcm(a,b)}lcm(a,b)n​，其中 lcmlcmlcm 为 最小公倍数。

四、源码剖析

#define ll long long 

int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);     // (1) 
}
ll lcm(int a, int b) {
    return (ll)a / gcd(a, b) * b;          // (2) 
}

int getABC(int n, int a, int b, int c) {   // (3) 
    ll ans = n / a + n / b + n / c - n / lcm(a, b) - n / lcm(b, c) - n / lcm(a, c) + n / lcm(lcm(a, b), c);
    return ans;
}
  • (1)(1)(1) 计算 aaa 和 bbb 的最大公约数；
  • (2)(2)(2) 计算 aaa 和 bbb 的最小公倍数；
  • (3)(3)(3) 利用容斥原理计算并集；
