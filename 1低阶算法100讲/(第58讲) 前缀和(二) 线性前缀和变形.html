一、概念定义

  有关前缀和的概念，在《算法零基础100讲》(第57讲) 前缀和(一) 线性前缀和入门 中已经较为清晰的阐述，今天我们来学习它的几个变种：前缀积 和 前缀异或和。

1、前缀积

  顾名思义，我们可以用一个prod[]数组来表示数组的前缀积，即prod[i]表示的是前 iii 项的积，数学公式如下：prod[i]=a[0]×a[1]×...×a[i]=∏k=0ia[k]prod[i] = a[0] \times a[1] \times ... \times a[i] = \prod_{k=0}^{i} a[k]prod[i]=a[0]×a[1]×...×a[i]=k=0∏i​a[k]  将 i−1i-1i−1 代入上述的 iii，得到：prod[i−1]=a[0]×a[1]×...×a[i−1]=∏k=0i−1a[k]prod[i-1] = a[0] \times a[1] \times ... \times a[i-1] = \prod_{k=0}^{i-1} a[k]prod[i−1]=a[0]×a[1]×...×a[i−1]=k=0∏i−1​a[k] 于是可以得到 prod[i]=prod[i−1]×a[i]prod[i] = prod[i-1] \times a[i]prod[i]=prod[i−1]×a[i]

2、前缀异或和

二、题目描述

1、定义

  数组 中心下标 是数组的一个下标，其 左侧所有元素相加的和 等于 右侧所有元素相加的和。

2、求解

  给你一个长度为 nnn 的整数数组 nums，返回输出数组 output，其中 output[i]等于 nums中除 nums[i]之外其余各元素的乘积。

三、算法详解

  直接用除法可能会产生溢出，所以我们采用 前缀积 乘上 后缀积 的形式计算每个数组的值。

四、源码剖析

int* productExceptSelf(int* nums, int numsSize, int* returnSize){
    int *pre = (int *)malloc( sizeof(int) * numsSize );    // (1) 
    int *post = (int *)malloc( sizeof(int) * numsSize );   // (2)
    int *ret = (int *)malloc( sizeof(int) * numsSize );    // (3)
    int i, cnt = 0;

    for(i = 0; i < numsSize; ++i) {                        // (4)
        pre[i] = nums[i];
        if(i) pre[i] *= pre[i-1];
    }
    for(i=numsSize-1; i >= 0; --i) {                       // (5)
        post[i] = nums[i];
        if(i<numsSize-1) post[i] *= post[i+1];
    }
    for(i = 0; i < numsSize; ++i) {                        // (6)
        if(i == 0) ret[i] = post[i+1];
        else if(i == numsSize-1) ret[i] = pre[i-1];
        else ret[i] = pre[i-1] * post[i+1];
    }
    free(pre);
    free(post);
    *returnSize = numsSize;
    return ret;
}
  • (1)(1)(1) 申请前缀积数组内存；
  • (2)(2)(2) 申请后缀积数组内存；
  • (3)(3)(3) 申请结果数组内存；
  • (4)(4)(4) 从前往后，初始化前缀积数组，时间复杂度 O(n)O(n)O(n)；
  • (5)(5)(5) 从后往前，初始化后缀积数组，时间复杂度 O(n)O(n)O(n)；
  • (6)(6)(6) 第 iii 元素的值为 i−1i-1i−1 的前缀积 乘上 i+1i+1i+1 的后缀积；
