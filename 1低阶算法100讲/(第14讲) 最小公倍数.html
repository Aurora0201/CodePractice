二、题目描述

  给定两个数 aaa、bbb，求它们的最小公倍数。答案保证不会超过 231−12^{31}-1231−1。

四、源码剖析

int gcd(int a, int b) {
    return !b ? a : gcd(b, a%b);  // (1)
} 

int lcm(int a, int b) {
    return a / gcd(a, b) * b;     // (2)
}
  • (1)(1)(1) 递归计算最大公约数；
  • (2)(2)(2) 先除法，再乘法，避免溢出；
