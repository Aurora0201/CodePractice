  两个数的最小值，利用C语言中的三元运算符就可以实现：

int Min(int a, int b) {
    return a < b ? a : b;
}

int Min(int a, int b) {                 // (1)
    return a < b ? a : b;
}

int findMin(int* nums, int numsSize){
    int i;
    int min = 100000001;                // (2)
    for(i = 0; i < numsSize; ++i)
        min = Min(min, nums[i]);        // (3)
    return min;
}

  • (2)(2)(2) 初始化候选最小数min为一个很大的数；
  • (3)(3)(3) 遍历数组，并且将候选最小数min和数组当前的数nums[i]进行比较，并且取小的那个，重新赋值给候选最小数min；
