一、概念定义

  什么是滚动哈希呢？
  当我求一个字符串的哈希值的时候，可以把这个字符串的前缀串的哈希值求出来，然后通过 O(1)O(1)O(1) 的差分，计算某个子串的哈希值。
  例如，对于字符串 sss 来说，用 h(i)h(i)h(i) 表示从第 000 个字符到第 iii 个字符串结束的前缀串的哈希值如下：

二、题目描述

  「快乐前缀」 是在原字符串中既是 非空 前缀也是后缀（不包括原字符串自身）的字符串。
  给你一个字符串 sss，请你返回它的 最长快乐前缀。如果不存在满足题意的前缀，则返回一个空字符串 “”。

三、算法详解

  (1)(1)(1) 如果我们能够快速获得一个子字符串的哈希值（比如用 O(1)O(1)O(1) 的时间复杂度来获取），那么对于长度为 xxx 的字符串，只要从 x−1x-1x−1 往下枚举前缀和后缀的长度，如果哈希值一致，直接返回就是答案了。
  (2)(2)(2) 如果找不到，返回空串。

四、源码剖析

class Solution {
    #define maxn 100010
    #define ull unsigned long long
    #define B 271
    ull power[maxn];
    ull hash[maxn];

    void init(string s) {               // (1)
        int i;
        power[0] = 1;
        hash[0] = s[0];
        for(i = 1; i < s.size(); ++i) {
            hash[i] = hash[i-1] * B + s[i];
            power[i] = power[i-1] * B;
        }
    }

    ull get(int l, int r) {             // (2)
        if(l == 0) {
            return hash[r];
        }
        return hash[r] - hash[l-1] * power[r-l+1];
    }
public:
    string longestPrefix(string s) {
        int i;
        init(s);
        for(i = s.size() - 1; i >= 1; --i) {  // (3)
            if( get(0, i-1) == get(s.size() - i, s.size() - 1)) {
                return s.substr(0, i);
            }
        }
        return "";
    }
};
  • (1)(1)(1) void init(string s)表示这个字符串的前缀哈希值。
  • (2)(2)(2) ull get(int l, int r)表示快速获取子串 s[l:r]s[l:r]s[l:r] 的哈希值，即 hash[r]−hash[l−1]∗Br−l+1hash[r] - hash[l-1] * B ^ {r-l+1}hash[r]−hash[l−1]∗Br−l+1
  • (3)(3)(3) 从大到小枚举长度 前缀 和 后缀 的长度，利用子串哈希来求解两个字符串是否相等。
