2、欧拉定理

【定义2】nnn 和 aaa 为正整数，且 nnn, aaa 互素，即 gcd(a,n)=1gcd(a,n) = 1gcd(a,n)=1，则：aϕ(n)≡1(mod n)a^{\phi(n)} \equiv 1 (mod \ n)aϕ(n)≡1(mod n)

  • 证明过程如下：
  • 1）令 1→n1 \to n1→n 中与 nnn 互素的数（总共 ϕ(n)\phi(n)ϕ(n) 个）按顺序排列为：
  • x1,x2,...,xϕ(n)x_1,x_2,...,x_{\phi(n)}x1​,x2​,...,xϕ(n)​
  • 2）将这些数分别乘上a，得到如下序列：
  • yi=axi (1≤i≤ϕ(n))y_i = ax_i \ (1 \le i \le \phi(n))yi​=axi​ (1≤i≤ϕ(n))
  • 3）取其中任意两个数相减，得到：
  • yi−yj=axi−axj=a(xi−xj)=ab\begin{aligned} y_i - y_j &= ax_i - ax_j \\ &= a(x_i - x_j) \\ &= ab\end{aligned}yi​−yj​​=axi​−axj​=a(xi​−xj​)=ab​
  • 上面的式子中 gcd(a,n)=1gcd(a, n)=1gcd(a,n)=1，且 ∣b∣=∣xi−xj∣<n|b| = |x_i - x_j| \lt n∣b∣=∣xi​−xj​∣<n，所以 ababab 不可能是 nnn 的倍数。从而得知 yiy_iyi​ 和 yjy_jyj​ 模 nnn 不同余。即 yi mod ny_i \ mod \ nyi​ mod n 有 ϕ(n)\phi(n)ϕ(n) 种余数。
  • 4）由欧几里得算法 (辗转相除法)，可以得到：
  • gcd(n,yi mod n)=gcd(yi,n)=gcd(axi,n)=1\begin{aligned} gcd(n, y_i \ mod \ n) &= gcd(y_i, n) \\ &= gcd(ax_i, n) \\ &= 1\end {aligned}gcd(n,yi​ mod n)​=gcd(yi​,n)=gcd(axi​,n)=1​
  • 所以得出，yi mod ny_i \ mod \ nyi​ mod n 与 nnn 互素；
  • 5）综合1）、3）和4），xix_ixi​ 和 yi mod ny_i \ mod \ nyi​ mod n 必然是一一映射。所以将所有的 xix_ixi​ 连乘，yi mod ny_i \ mod \ nyi​ mod n 连乘，两者必然模 nnn 同余。即：

  • 令 X=x1×x2×...×xϕ(n)X = x_1 \times x_2 \times ... \times x_{\phi(n)}X=x1​×x2​×...×xϕ(n)​，原式可以简化为：

  • 表示成更加直观的等式为：

三、欧拉函数的求解

1、素数试除法

【例题1】给定一个数 n(1≤n≤109)n(1 \le n \le 10^9)n(1≤n≤109)，求满足 gcd(i,n)=1(1≤i≤n)gcd(i, n)=1 (1 \le i \le n)gcd(i,n)=1(1≤i≤n) 的 iii 的个数。

1）算法原理

  • 根据欧拉函数的公式，只要一个数能够分解素因子，就能够求出它的欧拉函数；

所以算法原理如下：
  1）如果 n=1n = 1n=1，则直接返回 1；
  2）如果 n>1n > 1n>1，用 ansansans 来存储最终的欧拉函数值，初始值为 ans=1ans=1ans=1；
  3）然后，对所有 [2,n][2, \sqrt n][2,n​] 内的素数进行试除，对于素数 ppp，如果 p∣np|np∣n，则执行：n=n/pn = n / pn=n/p，ans=ans∗(p−1)ans = ans * (p-1)ans=ans∗(p−1)；如果这时候还是满足 p∣np|np∣n，则继续循环执行 n=n/pn = n / pn=n/p，每执行一次，则 ans=ans∗pans = ans * pans=ans∗p；
  4）当某个时刻，n=1n = 1n=1 时，直接返回 ansansans 的值；如果枚举完所有 [2,n][2, \sqrt n][2,n​] 内的素数， nnn 还是大于 1，表明这时候 nnn 是一个素数，返回 ans∗(n−1)ans * (n-1)ans∗(n−1) 即可；

2）算法实现

  • c++ 代码实现如下：
int Phi(int n) {
    if(n == 1)
        return 1;                  // 1)
    int ans = 1;                   // 2)
    for(int i = 0; i < ps; i++) {  // 3)
        int p = primes[i];
		if(n % p == 0) {
            n /= p;
            ans *= (p - 1);
            while( 0 == n % p )
                n /= p, ans *= p;
        }
        if(n == 1)                 // 4)     
            return ans;
    }
    return ans * (n - 1);
}
